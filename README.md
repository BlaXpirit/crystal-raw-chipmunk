# crystal-raw-chipmunk

#### [Crystal][] bindings to [Chipmunk][] (2D rigid body physics library).

Generated using [crystalize.py][], with manual modifications.

Requires Chipmunk 7.

[chipmunk]: http://chipmunk-physics.net/ "Chipmunk2D Physics"
